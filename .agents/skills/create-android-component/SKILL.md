# Інструкції для створення Android-компонента

Крок 1. **Аналіз вхідних даних**: Зрозумій, який функціонал має виконувати компонент.
Крок 2. **Імпорти**: Завжди імпортуй `View`, `Text`, `StyleSheet` з `react-native`. Для кнопок обов'язково використовуй `Pressable`.
Крок 3. **Специфіка Android**: 
- Додай `android_ripple={{ color: 'rgba(0, 0, 0, 0.1)' }}` до всіх інтерактивних елементів `Pressable`.
- Для тіней використовуй виключно властивість `elevation` (ігноруй `shadowOffset`, `shadowOpacity`, які працюють лише на iOS).
Крок 4. **Стилізація**: Створи константу `styles` за допомогою `StyleSheet.create({})` внизу файлу.
Крок 5. **Експорт**: Згенеруй повністю робочий, суворо типізований (TypeScript) компонент та експортуй його за замовчуванням. Не використовуй плейсхолдери.
